Adversarial Attacks Already Tell the Answer: Directional Bias-Guided Test-time Defense for Vision-Language Models

적대적 공격은 이미 답을 알려준다: 시각-언어 모델을 위한 방향성 편향 기반 테스트-시점 방어

Liangsheng Liu, Si Chen, Jiamin Wu, Weiwei Feng, Zhixin Cheng, Xiaotian Yin, Wenfei Yang, Tianzhu Zhang
인용 3
  • CLIP과 같은 시각-언어 모델(VLM)은 적대적 공격에 취약하여 실제 적용에 위험합니다.
  • 연구팀은 적대적 이미지가 VLM의 특징 공간에서 일관된 방향으로 이동하는 현상을 발견했습니다.
  • 이 현상을 바탕으로 방향성 편향 기반 방어(DBD)라는 테스트-시점 방어 프레임워크를 제안합니다.
  • DBD는 방어 방향(Defense Direction)을 추정하고 DB-점수 기반 재구성 전략으로 강건한 표현을 복구합니다.
  • 15개 데이터셋 실험에서 DBD는 최첨단 강건성을 달성하며, 적대적 정확도가 일반 정확도를 능가하기도 합니다.
TLDR (Semantic Scholar): Directional Bias-guided Defense (DBD), a test-time framework that estimates the Defense Direction and employs a DB-score-based two-stream reconstruction strategy to recover robust representations, is proposed, demonstrating that adversarial perturbations inherently encode directional priors about the true decision boundary.

Code2LoRA: Hypernetwork-Generated Adapters for Code Language Models under Software Evolution

Code2LoRA: 소프트웨어 진화 환경에서 코드 언어 모델을 위한 하이퍼네트워크 생성 어댑터

Liliana Hotsko, Yinxi Li, Yuntian Deng, Pengyu Nie
HF upvotes 72
  • 코드 언어 모델은 저장소(repository) 수준의 문맥이 필요하지만, 기존 방법은 비용이 많이 들고 코드베이스 변화에 취약합니다.
  • Code2LoRA는 하이퍼네트워크(hypernetwork)를 사용하여 저장소별 LoRA 어댑터를 생성, 추론 시 토큰 오버헤드 없이 저장소 지식을 주입합니다.
  • Code2LoRA는 정적 코드베이스용 Code2LoRA-Static과 코드 변경에 따라 업데이트되는 Code2LoRA-Evo 두 가지 시나리오를 지원합니다.
  • RepoPeftBench 벤치마크에서 Code2LoRA-Static은 정적 트랙에서 63.8%의 정확도를, Code2LoRA-Evo는 진화 트랙에서 60.3%의 정확도를 달성했습니다.
  • 이 프레임워크는 변화하는 코드 환경에 효과적으로 적응하는 코드 언어 모델을 위한 효율적인 지식 주입 방법을 제시합니다.
TLDR (Semantic Scholar): This work introduces Code2LoRA, a hypernetwork framework that generates repository-specific LoRA adapters, effectively injecting repository knowledge with zero inference-time token overhead, and evaluates Code2LoRA against parameter-efficient fine-tuning baselines.

Abhishek Indupally, Ibraheem Alawadhi, Satchit Ramnath, Jami J. Shah
매칭: neural network, data-driven, finite element, automotive, relevance:0.85

  • 본 연구는 설계 특징을 노드로, 관계를 엣지로 인코딩하는 속성 특징 그래프(AFG)를 제안합니다.
  • AFG는 CAD에서 파생된 특징 그래프에 직접 종단간 학습을 가능하게 합니다.

  • AFG는 설계 의도를 보존하며 엔지니어가 CAD에서 직접 수정하고 해석할 수 있게 합니다.

Learning finite viscoelasticity with DAVIS: A supervised framework for generalized standard materials

DAVIS를 이용한 유한 점탄성 학습: 일반화된 표준 재료를 위한 지도 학습 프레임워크

Simon Wiesheier, Paul Steinmann, Miguel Angel Moreno-Mateos
매칭: data-driven, finite element, elastic, relevance:0.80
  • 이 연구는 일반화된 표준 재료(generalized standard materials) 설정 내에서 유한 점탄성(finite viscoelasticity)을 위한 DAVIS 프레임워크를 개선합니다.
  • 기존 DAVIS의 비평형(non-equilibrium) 가지의 견고성과 식별 가능성을 높이기 위해 두 가지 확장을 제안합니다.
  • 첫째, 스플라인(spline) 표현을 곡률 기반 변수로 재구성하여 단조성 및 볼록성 제약을 적용합니다.
  • 둘째, 보간(interpolation) 도메인 적응을 내부 매개변수 식별과 분리하는 전략을 사용합니다.
  • 이 방법은 균일한 단축 하중-제거 테스트를 통해 비평형 가지의 식별 가능성과 견고성을 평가합니다.
TLDR (Semantic Scholar): This work revisits the recently proposed data-adaptive viscoelasticity (DAVIS) framework, a spline-based formulation of finite viscoelasticity within the generalized standard materials setting and improves the robustness and identifiability of non-equilibrium branches in generalized Maxwell-type models.
