Operadic consistency: a label-free signal for compositional reasoning failures in LLMs

연산자 일관성: LLM의 조합적 추론 실패를 위한 레이블 없는 신호

Nathaniel Bottman, Yinhong Liu, Kyle Richardson
Allen Institute for Artificial Intelligence
인용 1영향력 인용 1
  • LLM의 추론 실패를 감지하는 기존 방법들은 질문 내 샘플링과 자체 평가에 기반합니다.
  • 이 연구는 연산자 이론(operad theory)을 활용하여, LLM의 직접 답변과 분해된 질문의 조합 답변이 일치해야 한다는 '연산자 일관성(OC)'을 제안합니다.
  • 4개의 다단계 질의응답(QA) 데이터셋에서 12개 LLM(4B~671B)을 대상으로 OC를 평가했습니다.
  • OC는 모든 데이터셋에서 정확도와 강한 상관관계(Pearson $r \in [0.86, 0.94]$)를 보였으며, CoT-SC보다 더 일관적입니다.
  • OC는 CoT-SC와 의미론적 엔트로피를 넘어 추가적인 정보를 제공하며, 선택적 예측 성능을 향상시킵니다.
TLDR (Semantic Scholar): At the per-question level, OC contributes information beyond CoT-SC and semantic entropy on every dataset (cluster-robust $p \leq 10^{-16}$ for the OC coefficient), and the conclusion is robust to additionally controlling for constructed decomposition-aware baselines.

EvoArena: Tracking Memory Evolution for Robust LLM Agents in Dynamic Environments

EvoArena: 동적 환경에서 견고한 LLM 에이전트를 위한 메모리 진화 추적

Jundong Xu, Qingchuan Li, Jiaying Wu, Yihuai Lan, Shuyue Stella Li, Huichi Zhou, Bowen Jiang, Lei Wang 외 6명
HF upvotes 113
  • 기존 LLM 에이전트 평가는 정적 환경을 가정하지만, 실제 환경은 동적 변화에 적응해야 합니다.
  • 이 간극을 해결하기 위해, 동적 환경 변화를 모델링하는 벤치마크 스위트 EvoArena를 제안합니다.
  • 또한, 메모리 진화를 구조화된 업데이트 이력으로 기록하는 패치 기반 메모리 패러다임 EvoMem을 제안합니다.
  • EvoArena에서 현재 에이전트의 평균 정확도는 39.6%로 저조하며, EvoMem은 성능을 평균 1.5% 향상시킵니다.
  • EvoMem은 메모리 내 증거 포착을 개선하여 신뢰할 수 있는 에이전트 배포에 기여합니다.
TLDR (Semantic Scholar): EvoArena is introduced, a benchmark suite that models environment changes as sequences of progressive updates across terminal, software, and social domains, and EvoMem is proposed, a patch-based memory paradigm that records memory evolution as structured update histories, enabling agents to reason about environmental evolution through changes in their memory.

REMAL: Residual Equilibrium Manifold Active Learning for Surrogate-Based Multidisciplinary Design Analysis

REMAL: 잔여 평형 매니폴드 능동 학습을 이용한 대리 모델 기반 다분야 설계 해석

Kail Yuan, Ashwin Renganathan
매칭: surrogate model, structural, design optimization, digital twin, relevance:0.85
  • 결합된 공학 시스템의 다분야 설계 해석은 각 분야의 변수가 일치하는 평형 상태 계산이 필요합니다.
  • REMAL은 결합 시스템을 위한 잔여 매니폴드 대리 모델링 프레임워크를 제안합니다.
  • 이 방법은 다중 작업 가우시안 프로세스 모델을 통해 잔여 매니폴드의 대리 모델을 학습합니다.
  • 위성, 공기역학 구조, 가스 터빈 모델 등 4가지 공학 시스템 벤치마크에서 평가되었습니다.
  • REMAL은 설계 공간 전반에 걸쳐 반복적인 고정점 평가가 필요할 때 비용 효율성을 보였습니다.
TLDR (Semantic Scholar): REMAL is introduced, a residual manifold surrogate modeling framework for coupled systems that learns a surrogate model of the joint residual manifold via multitask Gaussian process models and shows the cost effectiveness when repeated evaluations of the fixed point across the design space are necessary.

Federated Digital Twin With Spatiotemporal Learning and Postquantum Security for Robust IIoT Anomaly Detection

견고한 IIoT 이상 탐지를 위한 시공간 학습 및 양자 내성 보안을 갖춘 연합 디지털 트윈

Muhammad Usama Tanveer, S. Hassan, S. Belhaouari
매칭: deep learning, anomaly detection, digital twin, relevance:0.75
  • 산업용 사물 인터넷(IIoT)은 정교한 사이버 공격에 취약하며, 실시간 보안이 필요합니다.
  • 본 연구는 IIoT 환경을 위한 연합 디지털 트윈(federated digital twin) 기반의 시공간 이상 탐지 방법을 제안합니다.
  • 이 프레임워크는 디지털 트윈 모델링, 시공간 딥러닝(DL), 연합 학습(FL), 양자 내성 암호(PQC) 보안을 통합합니다.
  • RT-IoT2022 데이터셋에서 99.89%의 탐지 정확도와 65%의 통신 오버헤드 감소를 달성했습니다.
  • 제안된 방법은 높은 탐지 성능과 데이터 프라이버시를 제공하며, 미래 양자 공격에도 대비합니다.
